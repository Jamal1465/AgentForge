# AgentForge Submission Package

This folder contains the materials required to present AgentForge as a capstone project for the Kaggle × Google AI Agents Intensive Vibe Coding course.

AgentForge is submitted as a capability-first AI Software Engineering Operating System. The core platform does not depend on hardcoded roles such as Backend Agent, Frontend Agent, Database Agent, or DevOps Agent. Instead, the Planner requests capabilities, and the Agent Registry selects compatible plugins.

## Core Claim

AgentForge demonstrates a production-oriented agent platform with:

- plugin-first architecture,
- capability-based task routing,
- workflow orchestration,
- shared memory,
- MCP-ready tool gateway,
- security guardrails,
- evaluation gates,
- observability and telemetry,
- CLI/API/deployment interfaces.
